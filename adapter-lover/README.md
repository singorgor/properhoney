# 适配恋人 - 在线情感测评网站

## 项目简介

"适配恋人"是一个专业的情感适配测评网站，通过科学的五维度评估模型，帮助用户了解自己的情感需求，找到最合适的伴侣类型。

## 核心功能

### 🔬 专业测评
- **46道精心设计的测评题目**，涵盖五大情感维度
- **科学的评分算法**，包含反向计分题确保准确性
- **深度自我认知**，生成详细的个人情感画像

### 📊 五维度评估
- **S (Security)**: 情感安全感需求
- **A (Autonomy)**: 自主/空间需求
- **G (Growth)**: 共同成长/事业重视度
- **R (Reality)**: 现实务实程度
- **E (Emotion)**: 情绪共鸣/理性方式

### 💝 八大伴侣类型
1. 稳定港湾型
2. 并肩打拼型
3. 滋养照顾型
4. 灵魂对谈型
5. 自由同伴型
6. 理性拍档型
7. 现实靠谱型
8. 情绪共振型

### 📋 详细报告
- 个人情感画像雷达图
- 推荐伴侣类型深度分析
- 避雷类型识别
- 相处建议和识别信号

## 技术架构

### 前端技术栈
- **React 18** + **TypeScript** - 现代化的前端框架
- **Ant Design** - 企业级UI组件库
- **Styled Components** - CSS-in-JS样式解决方案
- **Chart.js** - 数据可视化图表库
- **React Router** - 单页面应用路由管理

### 核心算法
- **多维评分算法** - 基于5个维度的综合评估
- **类型匹配算法** - 计算用户与各伴侣类型的适配度
- **动态报告生成** - 根据用户特征生成个性化分析

## 测评流程

1. **开始测评** - 用户进入首页，点击开始测评
2. **答题过程** - 逐题回答46道单选题，支持返回上一题
3. **结果计算** - 前端实时计算五维度得分和类型匹配
4. **报告生成** - 生成详细的个性化测评报告
5. **结果分享** - 支持重新测评和打印报告

## 算法说明

### 评分机制
- 每道题目选项对应不同维度的分值变化
- 包含6道反向计分题防止惯性答题
- 最终分值标准化为0-100分制

### 匹配算法
- 计算用户五维得分与各类型理想配置的差异度
- 差异越小，适配度越高
- 生成完整的适配度排名和匹配分析
