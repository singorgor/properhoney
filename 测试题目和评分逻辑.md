# 适配恋人 - 测试题目和评分逻辑

**品牌理念：懂自己，才能遇见对的人**

## 一、总体说明

### 评估维度

适配恋人采用五维度评估模型：

S：情感安全感需求

A：自主 / 空间需求

G：共同成长 / 事业重视度

R：现实务实度（相对浪漫感受）

E：情绪共鸣 / 理性方式（高 = 更情绪共鸣，低 = 更理性）

每题：单选 4 选 1（A/B/C/D）

评分方式：

每个选项为某些维度 +2 / +1 / 0 / -1

最后把 40 题累加，得到每个维度的总分，再去匹配伴侣类型。

三、你实现时可以怎么做（简要提示）

定义维度初始值：

let S=0, A=0, G=0, R=0, E=0;


根据题号 + 选项映射加分：

用一个对象：scoreMap[题号][选项] = { S: +2, A: +1, ... }

累计完 40 题后：

每个维度你可以再做一次线性映射成 0–100，或者直接用原始分值。

然后用之前我们说的"和伴侣类型理想配置的差值"来选：

差值最小 → 主类型

差值最大 → 避雷类型

## 三、反向计分题设计（新增6题）

为了提高测评的准确性，防止惯性答题，特加入以下反向计分题：

### Q41（S维度反向题）
以下哪种说法最符合你的真实想法？

A. 我很少因为伴侣没有及时回复而感到不安（S-2）
B. 我觉得独立的感情比依赖的关系更健康（S-1, A+1）
C. 我不会过分关注伴侣的情绪变化（S-1）
D. 我认为过度在意对方的感受反而会伤害关系（S-2, E-1）

### Q42（A维度反向题）
对于"个人时间"这个概念，你更认同？

A. 我希望伴侣能随时参与我的生活安排（A-2, S+1）
B. 我觉得情侣之间不应该有太多私人空间（A-1）
C. 分享一切才是亲密关系的体现（A-2）
D. 我喜欢两个人时刻保持连接的状态（A-1, S 0）

### Q43（G维度反向题）
关于伴侣的"上进心"，你的真实看法是？

A. 我认为过度追求成长会让人变得焦虑（G-1, E+1）
B. 平静的生活比奋斗更重要（G-2, R+1）
C. 不是所有人都需要在事业上拼命（G-1）
D. 我更看重当下的生活质量，而非未来的成就（G-2）

### Q44（R维度反向题）
在选择伴侣时，你认为哪个因素最不重要？

A. 对方的经济基础和物质条件（R-2）
B. 在重大决策中的理性分析能力（R-1）
C. 对未来生活的详细规划（R-1）
D. 处理实际问题的执行力（R-2, G-1）

### Q45（E维度反向题）
关于情感表达，你更倾向于哪种方式？

A. 我觉得过多的情绪交流会消耗关系能量（E-2, R+1）
B. 理性沟通比情感共鸣更重要（E-1）
C. 我不太习惯分享内心感受（E-1, A+1）
D. 过度的共情反而会让关系变得复杂（E-2）

### Q46（综合反向题）
以下哪种关系模式最让你感到不适？

A. 两个人时刻保持高度连接（S-1, A-1）
B. 需要不断确认彼此感情的状态（S-2, E-1）
C. 对未来有过多的规划和期待（G-1, R-1）
D. 频繁深入的情感交流（E-2, A 0）

**反向题使用说明：**
- 这6道题随机分布在46题中，用户无法识别哪些是反向题
- 负分在最终计算时正常累加，起到平衡作用
- 如果用户在某些反向题上选择与之前题目矛盾的答案，说明该用户更认真地在思考问题
